# 🔬 Cervical Cancer Prediction using XGBoost

## 📖 Overview
This project focuses on the application of machine learning in the healthcare domain, specifically building a predictive model for cervical cancer. The notebook walks through the complete data science workflow, from importing and exploring the medical dataset to visualizing key trends, preparing the data, and ultimately training a robust Extreme Gradient Boosting (XGBoost) classifier.

## 🎯 Objective
The primary objective is to develop and evaluate an XGBoost classification model capable of predicting cervical cancer risk based on patient demographic and medical data. By effectively performing exploratory data analysis and data preparation, the model aims to achieve high predictive accuracy for early detection scenarios.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Machine Learning:** XGBoost, Scikit-Learn
* **Data Manipulation & Visualization:** pandas, NumPy, Matplotlib, Seaborn

## 🧠 Key Learnings & Skills Demonstrated
* **Problem Formulation:** Understanding the medical business case and framing the classification problem.
* **Exploratory Data Analysis (EDA):** Investigating the dataset to identify missing values, outliers, and feature distributions.
* **Data Visualization:** Creating insightful plots to uncover correlations between patient attributes and target variables.
* **Data Preparation:** Cleaning, scaling, and splitting the data appropriately for model training.
* **Model Training:** Implementing and tuning an Extreme Gradient Boosting (XGBoost) classifier.
* **Model Evaluation:** Testing the trained model's performance and interpreting the classification metrics.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/Cervical-Cancer-Prediction-Using-XGBoost.git

2. Navigate into the downloaded project folder:
   cd your-repo-name

3. Install the required dependencies:
   pip install pandas numpy scikit-learn xgboost matplotlib seaborn jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially to run the analysis and train the model.
